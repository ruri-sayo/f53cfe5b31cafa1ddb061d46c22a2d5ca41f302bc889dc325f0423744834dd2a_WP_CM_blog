## 🚀 技術スタック

| 技術 | 用途 |

| **React** | インタラクティブツール |

| **TypeScript** | 型安全 |
| **GitHub Pages** | ホスティング |
| **GitHub Actions** | CI/CD |

---

## 📁 ディレクトリ構成

```
/
├── src/
│   ├── components/     # Astro/Reactコンポーネント
│   ├── layouts/        # ページレイアウト
│   ├── pages/          # ルーティング対象ページ
│   ├── content/        # Markdownコンテンツ（記事・小説）
│   └── data/           # JSONデータファイル
├── public/             # 静的ファイル
├── dev/                # 開発・保守ドキュメント
└── dist/               # ビルド出力（git管理外）
```

---

## 🧞 コマンド

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# ビルドプレビュー
npm run preview
```
