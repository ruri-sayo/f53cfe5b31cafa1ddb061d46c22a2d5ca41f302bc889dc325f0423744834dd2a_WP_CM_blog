城上コードメモが **v2.2** にアップデートしました！🎉

このバージョンでは、ユーザビリティとデザインの大幅な改善を行い、季節のイベントを楽しめる新機能を追加しました。

---

## 主な変更点

### 🎨 デザイン改善

#### ヘッダー/フッターのリニューアル
- 新しいテーマカラー `#7278a8` を採用
- 統一感のあるモダンなデザインに刷新

#### コンテンツレイアウトの改善
- 記事などのコンテンツを中央配置の左揃えに変更
- 読みやすさを向上させました

---

### 📱 モバイル対応の強化

#### ハンバーガーメニューの実装
- スマートフォンでもサイト内を快適に移動できるようになりました
- スムーズなスライドアニメーション付き

#### カード全体がクリック可能に
- 記事やツールのカードは、タイトルだけでなくカード全体をタップして移動できます

### 🎉 イベントモードの実装（新機能！）

季節やイベントに応じて、サイトのデザインが自動で変わります！

#### 対応イベント

| イベント | 期間 |
|---------|------|
| 🎍 正月 | 1/1 - 1/4 |
| 👹 節分 | 2/3 |
| 💝 バレンタイン | 2/14 |
| 🤡 エイプリルフール | 4/1 |
| 🎏 端午の節句 | 5/5 |
| 🎋 七夕 | 7/7 |
| 🏮 お盆 | 8/13 - 8/16 |
| 🌕 十五夜 | 9/10 |
| 🎃 ハロウィン | 10/31 |
| 🎄 クリスマス | 12/24 - 12/25 |

#### 開発者向け：デバッグコマンド
ひと足早くイベントモードを体感したい場合、ブラウザのコンソールで以下のコマンドを使用できます：

```javascript
// イベントを手動で設定
window.setEvent('christmas');

// 利用可能なイベント一覧を表示
window.listEvents();

// イベントをクリア
window.setEvent(null);
```
