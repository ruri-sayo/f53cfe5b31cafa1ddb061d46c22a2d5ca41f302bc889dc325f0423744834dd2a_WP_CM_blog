# このサイトについて

**「城上コードメモ」**へようこそ！

当サイトは、個人の感想や主張の記事、便利なWebツール、ブラウザゲーム、そして創作小説などを公開している個人ブログサイトです。
日々のプログラミング学習の備忘録（Code Memo）としても活用しています。

## コンテンツ紹介

当サイトでは主に以下のコンテンツを提供しています。

### 🛠️ Webツール
日常や開発に役立つツールをブラウザ上で無料で利用できます。
- **文字数カウント**: 入力したテキストの文字数をカウントします。
- **Diff比較**: 2つのテキストの差分を表示します。
- **スキー場標高比較**: スキー場の標高データを比較できるニッチなツールです。

### 🎮 ブラウザゲーム
インストール不要ですぐに遊べるゲームです。
- **大富豪**: ローカル対戦や、強化されたAIとの対戦が楽しめます。
- **タイピングゲーム**: 独自のストーリーやモードを搭載したタイピング練習ゲームです。

### 📝 技術記事・ブログ
- **技術備忘録**: C#、Git、生成AI（LLM）など、興味のある技術分野についての学習記録や解説。
- **PC基礎知識**: ハードウェア・ソフトウェアに関する基礎的な解説。
- **雑記**: 日々の出来事や、個人の考え（お祝い、お酒の話など）も時々綴ります。

### 📚 小説
創作カテゴリー（Novel）にて、オリジナルの小説作品を公開しています。
