# 運用・デバッグマニュアル

## 1. ヒーローエリアの切り替え方法

トップページ上部のヒーローエリア（重要なお知らせや特集への誘導バナー）は、以下のJSONファイルを編集することで制御できます。

**設定ファイルパス:** `src/data/hero.json`

### 設定項目
```json
{
  "enabled": true,            // 表示するかどうか (true/false)
  "title": "タイトルテキスト",
  "message": "本文テキスト",
  "bg_class": "Tailwind CSSクラス", // 背景グラデーションなど
  "button": {
    "enabled": true,          // ボタンを表示するかどうか
    "text": "ボタンのラベル",
    "url": "/link/to/page/"   // リンク先パス
  }
}
```

変更を保存すると、Webサイトに即座に反映されます（開発サーバー稼働中）。

## 2. 季節イベントモードのデバッグコマンド

サイトには日付に応じて自動的にテーマ（配色や絵文字）が切り替わる「イベントモード」が搭載されています。
開発者ツール（Console）を使用して、強制的にイベントを適用してテストすることができます。

### イベントデータ定義
**設定ファイルパス:** `src/data/events.json`

### デバッグ用コンソールコマンド

ブラウザのF12キーなどで開発者ツールを開き、コンソールタブで以下のコマンドを実行してください。

#### 利用可能なイベント一覧を表示
```javascript
window.listEvents()
```
現在定義されているすべてのイベントIDと期間が出力されます。

#### イベントを強制適用
```javascript
window.setEvent('event_id')
```
（例）クリスマステーマを適用する場合：
```javascript
window.setEvent('christmas')
```

#### イベントモードを解除（デフォルトに戻す）
```javascript
window.setEvent(null)
```
または引数なしで実行：
```javascript
window.setEvent('')
```

### 注意事項
- このコマンドによる変更は、ページをリロードするとリセットされます。
- `window.SHIROGAMI_EVENTS` グローバル変数にイベントデータが格納されています。
